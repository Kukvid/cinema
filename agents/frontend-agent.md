## Agent : Frontend Developer (Next.js)

**Технологии**: Next.js 14, React, Material-UI, Context API, Axios, React Hook Form

### Основные задачи

**Фаза 1: Аутентификация**
1. Настроить Next.js проект с App Router
2. Создать AuthContext с JWT токенами
3. Страницы login/register с React Hook Form
4. Protected routes для админ-панели
5. Profile page с отображением ролей

**Фаза 2: Публичные страницы**
6. Главная страница: каталог фильмов с фильтрами (жанр, рейтинг)
7. Страница фильма: детали + расписание сеансов по кинотеатрам
8. Выбор кинотеатра и даты сеанса
9. Интерактивная схема зала (grid seats с цветами: свободно/занято/проход)
10. Добавление товаров кинобара в заказ

**Фаза 3: Оформление заказа**
11. Страница checkout: резюме заказа (билеты + кинобар)
12. Применение промокода
13. Применение бонусов
14. Mock оплата (выбор способа оплаты)
15. Страница успешной оплаты с QR кодами билетов

**Фаза 4: Личный кабинет**
16. Мои билеты (активные/прошедшие) с QR кодами
17. История заказов
18. Бонусный счет с историей транзакций
19. Предзаказы кинобара с кодами выдачи
20. Редактирование профиля

**Фаза 5: Админ-панель**
21. Dashboard: статистика продаж
22. CRUD для кинотеатров, залов, мест
23. CRUD для фильмов (с загрузкой постеров)
24. CRUD для дистрибьюторов
25. Управление договорами проката (с процентами по неделям)
26. Управление сеансами с календарем (react-big-calendar)
27. CRUD для товаров кинобара
28. Управление промокодами
29. Просмотр и валидация билетов (сканирование QR)
30. Генерация и скачивание отчетов

**Best practices**:
- Context API: AuthContext, CartContext для корзины билетов
- Axios interceptors для добавления JWT токена
- Error boundaries для обработки ошибок
- Loading states для всех асинхронных операций
- Оптимистичные обновления UI где возможно
